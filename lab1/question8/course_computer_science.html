B.Sc. Computer Science

Course Description

The Bachelor of Science in Computer Science is a 3-year undergraduate program designed to provide students with a strong foundation in computing principles, software development, algorithms, and data structures. Students will gain hands-on experience with modern programming languages like Python, Java, and C++, as well as web technologies and database management systems.

Core Outcomes: Software Engineering proficiency, System Analysis, and Database Administration.

Faculty Members

  • Dr. Alan Turing Algorithms & AI
  • Prof. Ada Lovelace Software Engineering
  • Mr. Linus Torvalds Operating Systems
  • Mrs. Grace Hopper Compilers
